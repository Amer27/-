# خطوط التطبيق

## الخطوط المطلوبة:
- Tajawal-Regular.ttf
- Tajawal-Bold.ttf

## طريقة إضافة الخطوط:
1. نزل خطوط Tajawal من [هنا](https://fonts.google.com/specimen/Tajawal)
2. ضع الملفات في هذا المجلد
3. تأكد من الإعدادات في `pubspec.yaml`
